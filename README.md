JSON record version control for PHP

## Installation

### Library

```bash
git clone git@github.com:aaronbullard/paper-trail.git
```

### Composer

[Install PHP Composer](https://getcomposer.org/doc/00-intro.md)

```bash
composer require aaronbullard/paper-trail
```

### Testing

```bash
composer test
```
